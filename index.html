UWPX the Open Source XMPP app for your UWP devices

UWPX is a app for your UWP (Windows 10) devices. It implements the Extensible Messaging and Presence Protocol (XMPP). At the moment UWPX is still in ALPHA state so expect regular crashes and unexpected behaviour. If you encounter bugs or eaven crashs and somehow can reproduce them please report them over at GitHub.
Here your find an up to date list of all features and implemented XEPs.

UWPX is completely Open Source. It's licesed under Mozilla Public License 2.0. Feel free to clone, change and submit features and bug fixes - there are always bugs waiting to be fixed.

By default every connection between you and your server is encrypted via TLS 1.2. This prevents attackers from reading your chat messages or metadata.
OMEMO:
OMEMO support is actively beeing worked on. Here you can find more up to date information about the current OMEMO implementation state. Once implemented, OMEMO will provide a multi-client end-to-end encryption so not even your own server can read your chat messages any more.

You are not limited to normal chats with only one recipient. UWPX supports XEP-0045: Multi-User Chat (MUC). Just click "Add Group Chat (MUC)", enter the room name and voila you created a new or joind an existing MUC.
Still not enough?
You even can manage your rooms - change the room name, kick/ban members, set a password, ...

There comes the point where text alone is not enough. You need something more meaningful - like an image.
No problem we got you!
Just grab a link of an image send it and UWPX will do the rest for you.
UWPX supports from XEP-0363: HTTP File Upload at the moment only the receiving portion. There's more to come!
